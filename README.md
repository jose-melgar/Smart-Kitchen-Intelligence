# Smart Kitchen Intelligence (SKI)

## Descripción del Proyecto
**Smart Kitchen Intelligence (SKI)** es un proyecto técnico desarrollado para el curso de Big Data que busca transformar la gestión de alimentos en el hogar. El sistema funciona como un motor de descubrimiento y recomendación que integra el monitoreo de múltiples áreas de almacenamiento (refrigeradores, estanterías y despensas).

El proyecto combina metadatos reales de productos obtenidos a través de la **API de OpenFoodFacts** con datos simulados de interacción de usuario para generar análisis de salud, alertas de conservación y recomendaciones personalizadas.

## Objetivos Técnicos
Siguiendo las directrices del curso, este proyecto implementa un pipeline completo que abarca:
* **Capa de Catálogo:** Gestión de entidades de productos basadas en información nutricional real.
* **Ingeniería de Características:** Representación numérica de macros, nutrientes y ciclos de vida útil.
* **Analítica de Agrupamiento (Clustering):** Segmentación de perfiles de consumo y tipos de dieta.
* **Sistema de Recomendación:** Motor híbrido para sugerencia de recetas y optimización de inventario.
* **Inteligencia de Grafos:** Análisis de co-ocurrencia y sustitución de alimentos mediante métricas de centralidad y PageRank.

## Estructura del Repositorio
El repositorio sigue un flujo de trabajo reproducible, separando los datos crudos de los procesados y el código fuente de la experimentación en notebooks.
