# Nota de Ética y Acceso

## Origen y Permisos
- Los datos de productos provienen de **OpenFoodFacts**, una base de datos abierta y colaborativa protegida por ODbL.
- No se realiza scraping de perfiles privados ni acceso a datos restringidos.

## Privacidad de Datos Personales
Para mitigar riesgos de privacidad, **no se utilizan datos de usuarios reales**. Toda la capa de interacción (hábitos de consumo) es generada de forma sintética mediante simulación estadística, cumpliendo con las restricciones del curso sobre el manejo de información sensible.
